# Counter App

A simple and interactive Counter App built using HTML5, CSS3, and Vanilla JavaScript.

## Features

- Increase counter
- Decrease counter
- Reset counter
- Hold the Increase or Decrease button for continuous counting
- Counter color changes:
  - 🟢 Green for positive numbers
  - 🔴 Red for negative numbers
  - ⚫ Black for zero

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## JavaScript Concepts Used

- Variables
- Functions
- DOM Manipulation
- Event Listeners
- if...else
- setInterval()
- clearInterval()
